# AI Generative Prompts

This file documents the core prompts and instructions utilized to generate the architectural and visual structure of Nandri Connect using the Google AI Studio coding agent.

## Initial Concept Prompt
> "Build a mobile-first PWA for an NGO in India called Nandri Kinderhilfe. It should have a real-time feed of updates from sponsored children, a CRM dashboard for the local staff to manage students, and role-based views. Use Tailwind CSS and React."

## CRM Dashboard & Excel Interop Prompt
> "I need the admin dashboard to be able to handle bulk updates. Add import and export functionality using SheetJS (xlsx). Make sure there are tables with sticky headers, search inputs, and inline editing capabilities."

## Localization & Internationalization Prompt
> "Add a LanguageContext. The app needs to be fully localized in English and German since the sponsors are mostly based in Germany. Create a translation dictionary and toggle for 'EN' and 'DE'."

## Authentication & Role Segregation Prompt
> "Create a fully functional authentication UI (Login/Sign up) that uses local storage to maintain session state for an open-source deployment. Set `sthanna@salesforce.com` and `thannasudhir9@gmail.com` as default admin accounts with employee rights."
